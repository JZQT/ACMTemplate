# ACM_Template


## 概述
我的ACM代码模板知识库项目，包含**文档**和**代码**两类。供**学习**和**总结**之用。

* 文档

    包含算法讲解，心得体会以及一些说明等。

* 代码

    算法模板的代码实现。
