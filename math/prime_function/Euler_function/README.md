# 欧拉函数

## 概述

**欧拉函数**，也称为**欧拉phi函数**。

**对于正整数n，欧拉函数是不大于n的正整数中与n互质的数的个数。**

**欧拉函数的公式**是

> **phi(n) = n * (1 - 1/p1) * (1 - 1/p2) * (1 - 1/p3) * ··· * (1 - 1/pk)**

其中p1, p2, ··· , pk为n分解出的不同的质因数。

**欧拉函数的求值定义**是

* phi(1) = 1
* 如果n是素数p的k次幂，那么 **phi(n) = p^k - p^(k-1) = (p-1) * p^(k-1)**
* 如果m与n互质，那么 **phi(m*n) = phi(m) * phi(n)**

**ACM竞赛中的欧拉函数的求法**有两种

* 求单个数的欧拉函数值

    直接上面的求值定义来求

* 求欧拉函数表

    筛素数的同时计算出每个数的欧拉函数
